Dotfiles is a comprehensive configuration system that transforms your terminal into a powerful, modern development environment. With automated installation scripts and carefully curated tool integrations, you can set up a beautiful, productive shell environment in minutes.

## Key Features

<AccordionGroup>
  <Accordion title="Automated Installation" icon="wand-magic-sparkles">
    The `bootstrap.sh` script handles everything automatically - from installing dependencies to syncing configuration files. Run it once and your environment is ready.
  </Accordion>

  <Accordion title="Modern CLI Tools" icon="screwdriver-wrench">
    Pre-configured integration with the best modern CLI tools: `eza` for beautiful file listings, `bat` for syntax-highlighted file viewing, `atuin` for magical shell history, and Powerlevel10k for a stunning prompt.
  </Accordion>

  <Accordion title="Comprehensive Zsh Configuration" icon="scroll">
    Includes Oh My Zsh with carefully selected plugins, custom aliases and functions, and modular configuration snippets that load automatically.
  </Accordion>

  <Accordion title="Git Enhanced" icon="code-branch">
    Custom Git aliases for better productivity, beautiful log formatting, and helpful shortcuts for common workflows like branch management and PR creation.
  </Accordion>

  <Accordion title="Modular Scripts" icon="cube">
    Each tool has its own installation and upgrade script. Install only what you need, or update individual components without touching the rest.
  </Accordion>

  <Accordion title="Cross-Platform Support" icon="globe">
    Designed to work seamlessly on both macOS (with Homebrew) and Linux systems, with platform-specific optimizations where needed.
  </Accordion>
</AccordionGroup>

## Quick Start

Get started with a single command:

```bash
git clone https://github.com/ombratteng/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./bootstrap.sh
```

The bootstrap script will guide you through the installation process, installing dependencies and syncing your configuration files.

## What's Included

This dotfiles configuration includes:

- **Zsh Configuration**: Complete setup with Oh My Zsh, Powerlevel10k theme, and useful plugins
- **Modern CLI Tools**: eza, bat, atuin, yq, step, and more
- **Git Configuration**: Aliases, custom log formats, and productivity shortcuts
- **Tmux Setup**: Sensible defaults and key bindings
- **Vim Configuration**: Basic but functional editor setup
- **macOS Integration**: Brewfile for package management and system defaults
- **Custom Functions**: Shell functions for common tasks and workflows
