## What are Dotfiles?

Dotfiles are configuration files that customize your development environment. They're called "dotfiles" because they typically start with a dot (`.`) and are hidden by default on Unix-based systems. These files control everything from your shell prompt to your Git aliases to your terminal colors.

This dotfiles repository provides a pre-configured, batteries-included setup that transforms your terminal into a powerful, beautiful, and efficient development environment.

## How It Works

The dotfiles use a smart bootstrap script that:

1. **Installs dependencies** - Automatically installs Oh My Zsh, Powerlevel10k, and modern CLI tools
2. **Syncs configurations** - Uses rsync to copy dotfiles to your home directory
3. **Handles updates** - Provides commands to upgrade all tools and sync changes
4. **Preserves customizations** - Uses `.local` files for personal settings that won't be overwritten

<Note>
The bootstrap script is interactive by default, asking for confirmation before each installation step. Use the `--force` flag for unattended installation.
</Note>

## Key Features

### Intelligent File Sync

The sync process intelligently excludes certain files and includes others based on your platform:

```bash bootstrap.sh
rsync \
	--exclude ".git/" \
	--exclude "scripts/" \
	--exclude ".DS_Store" \
	--exclude "bootstrap.sh" \
	--include ".local/" \
	--exclude "*.local" \
	-avh --no-perms . ~;
```

### Local Customization

Dotfiles that end in `.local` are excluded from syncing, allowing you to maintain personal settings:

- `.zshrc.local` - Personal Zsh configuration
- `.zshenv.local` - Environment variables
- `.gitconfig.local` - Git user information

<Tip>
Use `.local` files to store sensitive information like API keys or personal preferences that shouldn't be synced from the repository.
</Tip>

### Modern Command Aliases

Once installed, traditional commands are automatically replaced with modern alternatives:

- `ls` → `eza` (modern file listing with Git status)
- `cat` → `bat` (syntax-highlighted file viewing)
- `df` → `duf` (disk usage with better formatting)
